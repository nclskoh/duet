val abstract : ?⁠exists:(Srk.Syntax.symbol -> bool) -> (module Domain with type t = 'a) -> C.t Srk.Syntax.formula -> 'a

Compute the best abstraction of a formula within a given domain. For a formula phi, this is an element of the abstract domain elt such that: (1) phi |= formula_of elt, (2) elt is expressed only over the symbols that satisy the exists predicate, and (3) for any elt' satisfying (1) and (2), we have elt <= elt'.
