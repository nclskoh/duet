Module Ring.MakeUltPeriodicSeq

Ultimately periodic sequences. An ultimately periodic sequence is an infinite sequence consisting of a finite transient phase and an infinite periodic phase (with finite period). Ring operations are pointwise:

  • (f+g)(i) = f(i) + g(i)
  • (f*g)(i) = f(i) * g(i) (Hadamard product)
  • (-f)(i) = -f(i)
  • (0)(i) = 0
  • (1)(i) = 1
