val solve_rec : ?⁠initial:QQ.t -> QQ.t -> t -> t

solve_rec ?initial lambda g computes an exponential-polynomial g such that

  • g(0) = initial
  • g(n+1) = lambda*g(n) + f(n)

. If the initial value is omitted, it is taken to be 0.
