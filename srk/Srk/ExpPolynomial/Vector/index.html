val of_enum : (scalar * dim) BatEnum.t -> t

Convert an enumeration into a vector. Satisfies of_enum (enum v) = v, but not necessarily enum (of_enum e) = e; equivalent to add_term a i over all (a,i) in the given enumeration so (1) zero entries are suppressed and (2) if the enumeration contains multiple entries for the samae position, those entries are added.

val of_list : (scalar * dim) list -> t

val pivot : dim -> t -> scalar * t

pivot i v find the entry at the ith position of v, and the result of removing the ith position (replacing it with 0). If (a,v') = pivot i v, then add_term a i v' = v.
