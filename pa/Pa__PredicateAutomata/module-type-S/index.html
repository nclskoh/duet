make alphabet vocabulary initial accepting creates a new PA with alphabet alphabet, vocabulary vocabulary, initial formula initial, and accepting predicates accept. Initially, the transition rule is defined to be delta(q,a) = false for all (q,a). Transitions can be added using add_transition.

val add_transition : t -> predicate -> letter_set -> formula -> unit

add_transition pa q letters phi destructively updates the PA pa by adding (for each letter in letters) the transition rule delta(q,letter) = phi \/ psi, where psi was the previous transition rule for (q,letter).

val conjoin_transition : t -> predicate -> letter_set -> formula -> unit

conjoin_transition pa q letters phi destructively updates the PA pa by adding (for each letter in letters) the transition rule delta(q,letter) = phi /\ psi, where psi was the previous transition rule for (q,letter).
